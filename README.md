안드로이드 스튜디오를 이용한 어플리케이션 개발 (2023.12.10~ 2024.1.13)

이 프로젝트에서는 기프티콘 관리 어플을 구현한다. 기능

기프티콘의 정보 (이미지/이름/유효기간)을 입력하여 sqlite에 저장한다. 저장된 기프티콘을 볼 수 있게 한다. 유효기간이 지나기 전에 알림이 오게 한다.
